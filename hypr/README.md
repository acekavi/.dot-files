## üé® macOS Theme Features

### Visual Design
- **macOS-inspired color scheme** with Material Design 3 integration
- **Subtle transparency and blur effects** similar to macOS windows
- **Rounded corners** with 12px radius for authentic macOS feel
- **Drop shadows** positioned like macOS windows

### Keyboard Shortcuts (macOS-style)
- `Super + Space` - Spotlight search (DankShell launcher)
- `Super + Q` - Quit application (like Cmd+Q)
- `Super + W` - Close window (like Cmd+W)
- `Super + T` - New terminal (like Cmd+T)
- `Super + N` - New browser window (like Cmd+N)
- `Super + L` - Lock screen (like Cmd+L)
- `Super + ,` - System preferences (like Cmd+,)
- `Ctrl + Left/Right` - Switch workspaces (like macOS Spaces)
- `Alt + Tab` - Application switcher

### Window Management
- **Dwindle layout** for automatic window tiling
- **Smart gaps** that adapt to window count
- **Floating windows** for dialogs and utilities
- **Mission Control style** workspace overview
- **Picture-in-Picture** support for media

## üîß Configuration

### Customizing Applications
Edit `programs.conf` to set your preferred applications:
```bash
$terminal = alacritty    # Your terminal emulator
$browser = brave         # Your web browser
$fileManager = nautilus  # Your file manager
$editor = cursor         # Your text editor
```

### Monitor Setup
Edit `monitors.conf` for your display configuration:
```bash
# Primary laptop display
monitor = eDP-1, 2240x1400@60, auto, 1.46

# External monitor (uncomment and modify)
# monitor = DP-1, 1920x1080@144, 0x0, 1
```

### Color Theming
Colors are automatically generated by matugen and stored in `colors.conf`. To change the theme:

```bash
# Apply a preset color theme
./scripts/apply-macos-theme.sh --preset blue --reload

# Use a custom color
./scripts/apply-macos-theme.sh "#FF2D92" --reload

# Extract color from wallpaper
./scripts/apply-macos-theme.sh --wallpaper --reload

# Generate light theme
./scripts/apply-macos-theme.sh --preset green --light --reload
```

### Available Color Presets
- `blue` - macOS Blue (default)
- `purple` - macOS Purple
- `pink` - macOS Pink
- `red` - macOS Red
- `orange` - macOS Orange
- `yellow` - macOS Yellow
- `green` - macOS Green

## üöÄ Quick Setup

1. **Backup your current configuration** (already done):
   ```bash
   cp hyprland.conf hyprland.conf.backup
   ```

2. **Apply a theme**:
   ```bash
   ./scripts/apply-macos-theme.sh --preset blue --reload
   ```

3. **Customize applications** in `programs.conf`

4. **Adjust monitor settings** in `monitors.conf`

5. **Reload configuration**:
   ```bash
   hyprctl reload
   ```

## üìù Customization Tips

### Adding Custom Rules
Add custom window or workspace rules to the bottom of `hyprland.conf`:
```bash
# Custom rule example
windowrulev2 = workspace 5, class:^(my-app)$
```

### Custom Keybindings
Add custom shortcuts to `keybinds.conf`:
```bash
# Custom keybinding example
bind = SUPER, F1, exec, my-custom-script
```

### Performance Tuning
Adjust animation and blur settings in `theme.conf` based on your hardware:
```bash
# For lower-end hardware, reduce blur
blur {
    size = 3
    passes = 1
}

# For high-end hardware, increase quality
blur {
    size = 8
    passes = 3
}
```
